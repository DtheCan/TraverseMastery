# Traverse Mastery 📐✨

**Лицензия:** [MIT](https://github.com/DtheCan/TraverseMastery/tree/main?tab=GPL-3.0-1-ov-file#)  

**Traverse Mastery** — это мобильное приложение на Flutter, предназначенное для:
- Инженеров-геодезистов
- Студентов
- Всех, кому необходимо выполнять расчеты замкнутых теодолитных ходов

Возможности приложения:
- Простой ввод данных измерений
- Точные расчеты невязок и координат
- Удобный интерфейс для специалистов
- Сохранение проводимых расчетов


---


## 🌟 Возможности

*   **Интуитивно понятный ввод данных:**
    *   ➕ Добавление и ➖ удаление станций.
    *   🏷️ Ввод пользовательских имен станций.
    *   🧭 Ввод измеренных горизонтальных углов в формате **Градусы° Минуты' Секунды"**.
    *   📏 Ввод расстояний между станциями.
    *   ✔️ Встроенная валидация данных для предотвращения ошибок.
*   **Автоматический расчет замкнутого теодолитного хода:**
    *   ∑ Расчет суммы измеренных и теоретической суммы углов.
    *   📉 Вычисление угловой невязки и ее сравнение с допустимой.
    *   ⚙️ Распределение угловой невязки и расчет исправленных горизонтальных углов.
    *   🗺️ Расчет дирекционных углов.
    *   ➡️ Расчет приращений координат (ΔX, ΔY).
    *   ❗️ Вычисление линейных невязок по осям и абсолютной/относительной невязок.
    *   ✅ Сравнение линейных невязок с допустимыми значениями.
    *   📍 Расчет координат точек хода.
*   **Детализированное отображение результатов:**
    *   📊 Подробный отчет с результатами всех вычислений.
    *   🚦 Визуальное выделение допустимых (✅) / недопустимых (❌) невязок.
    *   🕰️ Форматированный вывод углов в ГГ°ММ'СС".
*   **Современный интерфейс:**
    *   ✨ Чистый, адаптивный и удобный дизайн.
    *   🌗 Поддержка системной светлой и темной тем оформления.

## 🛠️ Технологический стек  

* **[Flutter](https://flutter.dev/)** - UI toolkit для создания красивых, нативно компилируемых приложений для мобильных, веб и десктопных платформ из единой кодовой базы.  
* **[Dart](https://dart.dev/)** - Язык программирования, оптимизированный для разработки быстрых приложений на любой платформе.  

### 📦 Основные зависимости:  
* **[cupertino_icons](https://pub.dev/packages/cupertino_icons)** - Официальные иконки в стиле Cupertino (iOS) для Flutter.  
* **[uuid](https://pub.dev/packages/uuid)** - Генерация уникальных RFC4122 UUID.  
* **[intl](https://pub.dev/packages/intl)** - Интернационализация и форматирование (даты, числа, строки).  

### 📁 Работа с файлами и хранилищем:  
* **[path_provider](https://pub.dev/packages/path_provider)** - Доступ к путям файловой системы (временные, документы, кэш).  
* **[file_picker](https://pub.dev/packages/file_picker)** - Выбор файлов (включая множественный) с поддержкой всех платформ.  

### 🔐 Разрешения и безопасность:  
* **[permission_handler](https://pub.dev/packages/permission_handler)** - Запрос и проверка разрешений на Android/iOS.  

### 🔗 Интеграции и обмен данными:  
* **[share_plus](https://pub.dev/packages/share_plus)** - Нативный шеринг контента (текст, файлы) через системный диалог.
